# SEIA-Py
Simple Experimental Image Analyzer - Python  
SEIA는 간단한 실험 이미지 분석기입니다.  
교육과정 내 <a href="#Use">아래</a>와 같은 실험들을 분석하는데 사용할 수 있습니다. 

## 기능
1. 데이터 좌표 값
2. 데이터 길이 측정
3. 데이터 각도 측정
4. 데이터 색상 추출
5. 데이터 명도 그래프
6. 표 지우기 등

## <a name="Use">활용 예시</a>
- 물체의 운동 실험(속력, 등속/자유 낙하/등가속도/포물선/단진자 운동, 1차원/2차원 충돌)
- 파동의 성질 실험(빛의 반사/굴절/간섭/회절)
- 물질의 성질 실험(크로마토그래피, 지시약, 불꽃반응)
- 세포 현미경 관찰
- 광물과 암석의 관찰 등
